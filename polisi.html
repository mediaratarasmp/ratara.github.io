Privacy Policy

Kami menghargai privasi kamu. Kebijakan ini menjelaskan data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan bagaimana kami melindunginya.

Terakhir diperbarui: 1 Januari 2026

1. Data yang Kami Kumpulkan

Saat kamu menggunakan layanan Ratara SMP, kami dapat mengumpulkan data berikut:

  • Username Minecraft — digunakan untuk memberikan rank, item, dan memverifikasi identitas pemain di server.
  • Tipe Minecraft (Java/Bedrock) — digunakan untuk kompatibilitas akun di server.
  • Nomor WhatsApp — hanya jika kamu menghubungi Admin untuk konfirmasi transaksi atau laporan.
  • Data Transaksi — bukti pembayaran yang kamu kirimkan ke Admin untuk verifikasi pembelian.
  • Aktivitas Server — log aktivitas in-game seperti chat, posisi, dan tindakan pemain untuk keperluan moderasi.

2. Bagaimana Kami Menggunakan Data

Data yang kami kumpulkan digunakan semata-mata untuk:

  • Memproses dan memverifikasi pembelian di Ratara Store.
  • Memberikan rank, item, dan fitur premium yang telah dibeli.
  • Menjaga keamanan dan keadilan server melalui moderasi.
  • Merespons laporan dan pertanyaan dari pemain.
  • Meningkatkan pengalaman bermain di server.

Kami tidak menggunakan data kamu untuk keperluan iklan, pemasaran pihak ketiga, atau tujuan komersial lainnya.

3. Berbagi Data dengan Pihak Ketiga

Ratara SMP tidak menjual, menyewakan, atau membagikan data pribadi kamu kepada pihak ketiga manapun, kecuali:

  • Diwajibkan oleh hukum atau peraturan yang berlaku.
  • Diperlukan untuk melindungi hak dan keamanan server serta pemain lainnya.
  • Atas persetujuan eksplisit dari kamu sendiri.

4. Keamanan Data

Kami berkomitmen menjaga keamanan data kamu. Langkah-langkah yang kami ambil antara lain:

  • Data transaksi hanya bisa diakses oleh Admin yang berwenang.
  • Komunikasi dengan Admin dilakukan melalui platform terenkripsi (WhatsApp).
  • Log server disimpan secara aman dan hanya diakses untuk keperluan moderasi.

Namun perlu diketahui bahwa tidak ada sistem yang 100% aman. Kami akan memberitahu kamu jika terjadi kebocoran data yang berpotensi merugikan.

5. Penyimpanan Data

Data kamu disimpan selama kamu aktif sebagai pemain di Ratara SMP. Rinciannya:

  • Data akun & rank — disimpan selama akun aktif di server.
  • Log transaksi — disimpan selama 6 bulan setelah transaksi.
  • Log aktivitas server — disimpan selama 30 hari untuk keperluan moderasi.
  • Data laporan — disimpan hingga kasus selesai ditangani.

6. Hak Kamu

Sebagai pemain Ratara SMP, kamu memiliki hak untuk:

  • Mengakses data pribadi yang kami miliki tentang kamu.
  • Meminta koreksi jika ada data yang tidak akurat.
  • Meminta penghapusan data kamu dari sistem kami.
  • Menarik persetujuan penggunaan data kapan saja.

Untuk menggunakan hak-hak di atas, hubungi Admin kami melalui halaman laporan.

7. Cookie & Teknologi Pelacakan

Website Ratara SMP menggunakan cookie minimal yang diperlukan untuk fungsi dasar website. Kami tidak menggunakan cookie pelacakan, analytics pihak ketiga, atau iklan berbasis perilaku.

8. Pengguna di Bawah Umur

Ratara SMP tidak secara sengaja mengumpulkan data dari anak-anak di bawah 13 tahun. Jika kamu berusia di bawah 13 tahun, harap dapatkan izin dari orang tua atau wali sebelum menggunakan layanan kami. Jika kami mengetahui data anak di bawah umur tersimpan tanpa izin, kami akan segera menghapusnya.

9. Perubahan Kebijakan

Kami berhak memperbarui Privacy Policy ini sewaktu-waktu. Perubahan signifikan akan diumumkan melalui Discord server kami. Dengan terus menggunakan layanan Ratara SMP setelah perubahan, kamu dianggap menyetujui kebijakan yang baru.
